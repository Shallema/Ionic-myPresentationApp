Demo Demo 1 - Geolocation This plugin provides information about the device's location, such as latitude and longitude. This API is based on the W3C Geolocation API Specification. Demo 2 - OpenWeather API OpenWeatherMap Forecast API : 5 day forecast is available at any location or city. It includes weather data every 3 hours. Demo 3 - iRail API iRail supports digital creativity concerning mobility in Belgium. This is an attempt to make the railway time schedules in Belgium easily available for anyone. The iRail api allows anyone to query trains, stations, liveboards and connections.
